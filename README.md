# PYTHON-INTRODUCTION-SMOKING-PREVALENCE

## Overview

This repository contains the code and supporting materials for a group project completed as part of the course:

```
"Introduction to Data Analysis with Python"
```

The project focuses on analyzing smoking prevalence data using Python-based data analysis tools.

## Project Structure

The repository is organized as follows:

* data/
  Contains both raw and cleaned datasets used in the analysis.

* code/
  Contains the project code, organized into separate Jupyter notebooks for each task or analysis step.

* output/
  Stores figures, tables, and other outputs generated by running the code.

* requirements.txt
  Lists all Python packages required to run the project.
